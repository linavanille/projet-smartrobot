\subsection{Tests}
Nous n'avons pas réalisés de tests propres tels qu'on les a vus en cours avec Cunit. Mais pour chaque 
TAD nous avons réalisé un module de tests. Par exemple pour le TAD pile, nous avons implémenter un 
programme qui empile, dépile tout en affichant l'état de la pile. Ceci nous permet de nous assurer
que nos TAD fonctionnent avant de les utiliser dans la recherche de chemin.

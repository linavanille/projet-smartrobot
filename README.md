# SmartRobot - Robot autonome résolvant un labyrinthe

## Description
Ce projet a pour objectif de concevoir un robot autonome capable de sortir d’un labyrinthe.

Il s’inscrit dans une démarche pédagogique visant à approfondir des compétences en :
- algorithmique
- programmation en langage C
- électronique

Le projet met également l’accent sur le travail en équipe et les pratiques collaboratives, essentielles dans une formation d’ingénieur.



## Objectifs du projet
- Concevoir un robot autonome
- Exploiter des capteurs pour interagir avec l’environnement
- Développer des algorithmes de résolution de labyrinthe
- Générer et exécuter un chemin optimal vers la sortie



## Fonctionnement
Le robot utilise différents capteurs pour percevoir son environnement et se déplacer de manière autonome.

À partir d’un plan de labyrinthe fourni en entrée :
1. Un algorithme calcule un chemin vers la sortie
2. Ce chemin est traduit en instructions
3. Le robot exécute ces instructions pour sortir du labyrinthe



## Architecture du projet

### 1. Conception algorithmique
- Représentation du labyrinthe
- Algorithmes de recherche de chemin (ex: parcours, résolution)
- Génération d’instructions de déplacement

### 2. Conception électronique
- Intégration des capteurs
- Communication avec le robot
- Gestion des entrées/sorties



## Fonctionnalités principales
- Résolution automatique de labyrinthe
- Génération de trajectoire
- Exécution autonome par le robot
- Interaction avec l’environnement via capteurs



## Technologies utilisées
- Langage C
- Électronique embarquée
- Capteurs

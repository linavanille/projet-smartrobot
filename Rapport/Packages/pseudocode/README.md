# Comment utiliser le fichier .dtx ?

## Téléchargement
Dans un premier temps, il vous faudra télécharger le dossier pseudocode.
Ce dossier doit contenir :
 - le fichier pseudocode.dtx, composé du package et de sa documentation
 - le fichier pseudocode.ins qui vous permettra d'extraire le package .sty depuis le fichier .dtx
 - le fichier makefile, qui permet d'obtenir la documentation pdf et le package .sty en une seule commande

## Installation
Pour installer le package :
 1. Se positionner dans le dossier pseudocode
 2. ouvrir un terminal
 3. taper la commande suivante : make
 4. Vous vous retrouvez avec le package pseudocode.sty, ainsi que sa documentation pseudocode.pdf

## Utilisation
Il y a trois manières d'utiliser ce package dans votre document. Elles sont détaillées dans le pdf de documentation.
